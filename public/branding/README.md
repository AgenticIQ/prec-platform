# Branding Assets

Please place your branding assets in this folder:

## Required Assets:
- **logo.png** or **logo.svg** - Main company logo
- **logo-white.png** or **logo-white.svg** - White version for dark backgrounds
- **favicon.ico** - Browser favicon
- **og-image.png** - Social media preview image (1200x630px recommended)

## Optional Assets:
- **logo-icon.png** - Square icon version of logo
- Any additional brand assets (patterns, textures, etc.)

## Notes:
- SVG format preferred for logos (better scaling)
- PNG format should be high resolution (2x or 3x for retina displays)
- Keep file sizes optimized for web
